# TSTWEB — Aplicación web simple para pruebas

Descripción
------------
Aplicación que muestra y permite añadir estudiantes a diferentes ciclos formativos de formación profesional.
Aplicación web ligera creada para realizar pruebas y prototipos rápidos. Provee una interfaz mínima para verificar flujos, endpoints y comportamiento en el navegador.
Aglutina diferentes tecnologías para desarrollo de aplicaciones web.

Características
------------
- Interfaz web simple (HTML/CSS/JS).
- Endpoints básicos para pruebas (GET/POST).
- Código fácil de modificar y extender.
- Ideal para demos y pruebas locales.

Tecnologías
------------
- HTML, CSS, JavaScript, PHP

Cómo usar
------------
- Úsala como quieras, es para probar.
